
## SPIとは
SPI = Serial Peripheral Interface

以下の3本でデータ通信を行う
SDO = Serial Data Out
SDI = Serial Data In
SCK = Serial Clock


## I2Cとの違い
| 項目 | SPI | I2C |
|------|-----|-----|
| **信号線数** | 4本 | 2本 |
| **速度** | 高速 | 中速 |
| **配線** | 複雑 | シンプル |
| **デバイス数** | CS線で制限 | アドレスで多数接

## 構造

```
    マスター               スレーブ
   (マイコン)             (センサーなど)
   ┌─────────┐           ┌─────────┐
   │         │ ──SCK──→  │         │ ← クロック
   │         │ ──MOSI─→  │         │ ← データ送信
   │         │ ←─MISO──  │         │ ← データ受信
   │         │ ──CS───→  │         │ ← チップ選択
   └─────────┘           └─────────┘
```
